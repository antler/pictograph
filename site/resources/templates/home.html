Get Started.

Caribou was designed to remove the extraneous
effort usually required to develop a web application.
By focusing all effort into developing features that
can be reused over and over, you need only specify
what your application is supposed to do.

Hook Up The Admin

Caribou comes with an admin out of the box. You can find it here: caribou admin

Create Some Pages

Every page needs a template and a controller. Templates by default you the freemarker templating. Controllers are written in Clojure

Create Some Models

The Caribou admin easily lets you model out your data and create association across datatypes.

Add Content

With your models in place, it's time to start adding content!
